¿Cuál es la ventaja de utilizar getElementById frente a querySelector?

La ventaja de usar getElementById es que es más directo y rápido cuando ya sabemos el id exacto del elemento que queremos. Es más simple porque solo busca por id. En cambio querySelector es más flexible porque permite usar selectores de CSS, pero puede ser un poco más general.

¿En qué situación es preferible usar querySelectorAll en lugar de querySelector?
querySelector se usa cuando solo necesitamos seleccionar un elemento, porque devuelve el primer elemento que coincide con el selector.
